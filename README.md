 ## Setup
 
 To run the server, install it locally using npm:
 ```
 $ git clone https://github.com/krzysztofrusiecki/ad_recruitment_tasks.git
 $ cd ad_recruitment_tasks/z2
 $ npm install
 $ npm start
 ```
 To run the client, install it locally using npm:
 ```
 $ cd ../z3
 $ npm install
 $ npm start
 ```
